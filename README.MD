# NewsPortal1

Новостной портал на Flask с возможностью добавления новостей, просмотра по категориям и детального просмотра новостей.

## Описание

NewsPortal1 - это простое веб-приложение для управления новостями, построенное на фреймворке Flask. Приложение позволяет:
- Просматривать список новостей на главной странице
- Добавлять новые новости через форму
- Просматривать новости по категориям
- Просматривать детальную информацию о каждой новости

## Технологический стек

- **Backend**: Python 3.x, Flask 2.3.3
- **Frontend**: HTML5, CSS3, Bootstrap 5
- **База данных**: SQLite с SQLAlchemy ORM
- **Формы**: Flask-WTF, WTForms
- **Окружение**: python-dotenv для управления переменными окружения

## Требования

- Python 3.7 или выше
- pip (менеджер пакетов Python)

## Установка и запуск

1. Клонируйте репозиторий:
   ```bash
   git clone <url-репозитория>
   cd NewsPortal1
   ```

2. Создайте виртуальное окружение:
   ```bash
   python -m venv venv
   ```

3. Активируйте виртуальное окружение:
   - Для Windows:
     ```bash
     venv\Scripts\activate
     ```
   - Для Linux/MacOS:
     ```bash
     source venv/bin/activate
     ```

4. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```

5. Настройте переменные окружения:
   Скопируйте пример файла окружения и отредактируйте его:
   ```bash
   cp .env.example .env
   ```
   Или создайте файл `.env` в корне проекта с содержимым:
   ```
   DATABASE_URI=sqlite:///db.sqlite3
   SECRET_KEY=ваш_секретный_ключ_здесь
   ```

6. Выполните миграции базы данных (если применимо):
   ```bash
   flask db upgrade
   ```
   В данном проекте таблицы создаются автоматически при запуске приложения через `db.create_all()` в `app/__init__.py`.

7. Запустите приложение:
   ```bash
   python news.py
   ```
   Или:
   ```bash
   flask run
   ```

8. Откройте браузер и перейдите по адресу: http://127.0.0.1:5000/

## Переменные окружения

Создайте файл `.env` в корне проекта с следующими переменными:

- `DATABASE_URI` - строка подключения к базе данных (по умолчанию: `sqlite:///db.sqlite3`)
- `SECRET_KEY` - секретный ключ для защиты форм и сессий
